About the class

The purpose of the class is to learn how to make live websites from scratch using languages such as HTML, Java, and CSS. We learned how to make the website itself using HTML, add or change little details with CSS, and add things such as buttons search boxes with Java. We used GitHub to store the repositories adn to make our websites live.
