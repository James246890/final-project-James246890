GitHub




What is GitHub?

GitHub is a web-based Git or version control repository and Internet hosting service. It offers all of the distributed version control and source code management functionality of Git as well as adding its own features.

Why do we use GitHub?

We use Github because it is the "cloud" for code. GitHub hosts our source code projects in a variety of different programming languages, and keeps track of the changes made to every iteration. It is able to do this by using git, a revision control system that runs in the command line interface.
